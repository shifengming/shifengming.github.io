# 项目放在github上2

# 1，在github上创建一个新仓库

# 2，git clone

本地克隆刚才创建的项目

# 3，将本地的文件夹里面的所有内容拷贝到新克隆下来的文件夹中

# 4，cd 到克隆下来的文件

例：cd seller

# 5，git add .

是将本地文件添加到本地缓存区

# 6，git commit -am “注释”

引号里面的注释是你这次提交的什么内容做个注释，方便以后方便看

# 7，git push -u origin master

将本地缓存仓库的文件推送到远程，也就是网上

最后在Github上边看到的效果
